    populate_database.py(initial data collection for database)
    README.txt

Requirements: Python 2.7.10


To run the Item Catalog website, 
copy the catalog folder into the vagrant folder
for vagrant startup instructions see project information
execute 
python database_setup.py
python populate_database.py
python catalog.py

view item catalog web pages at http://localhost:8000
